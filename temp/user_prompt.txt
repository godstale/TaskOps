일주일 간의 주식 정보를 리서치해서 HTML 페이지로 만들고 싶다.
   - 일 단위의 KOSPI/KOSDAQ 차트,  주식 동향, 급등락주, 애널리스트 분석 등을 1페이지 분량으로 수집한다.
   - 자료 조사를 위해 신뢰할만한 사이트를 조사한다.
   - 위에 나열된 주제별로 자료를 수집한다.
   - 수집한 자료를 분석해서 리포트로 만든다.
   - 만들어진 리포트를 일일 단위 종합 페이지 + 상세 페이지의 HTML 네비게이션 구조로 만든다.
   - HTML 상단 드롭박스로 날짜를 선택해서 원하는 날짜의 종합 페이지로 이동 가능하도록 만든다.
   - 원하는 날짜만 입력하면 위 작업들이 다시 수행될 수 있도록 작업 계획을 작성한다.
   - 일단 03.16, 03.17 이틀간의 페이지부터 만든다.
